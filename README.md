# Installation And Usage

Follow these steps to set up and run the project:

1. **Clone the repository:**

   ```bash
   https://github.com/Mutah3r/mernifyU-todo-list.git
   ```

2. **Install dependencies:**

   ```bash
   cd mearnifyU-todo-list
   npm install
   ```

3. **Start the server:**

   ```bash
   npm start
   ```
